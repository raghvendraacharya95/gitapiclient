Description

	1. This Web application to display number of open issues based on the given git repository URL written in jQuery, JavaScript, Bootstrap.

Assumption

	1. Url is case sensitive.

Setup

	1. Content delivery network(CDN) used for jQuery and Bootstrap requires internet connectivity.

Run the application

	1. Open the index.html page in gives an URL.

    2. Enter the Url.

	2. Click on Fetch button.

Important

    1. Url must be complete and repository should be public (eg: https://github.com/owner/repository).
